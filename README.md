 # 🩺 Development of a Diabetes Risk Classification System Using Predictive Analytics

An AI-powered Machine Learning application that predicts whether a patient is at **Low Diabetes Risk** or **High Diabetes Risk** using health parameters from the Pima Indians Diabetes Dataset.

---

## 📌 Project Overview

This project uses the **Random Forest Classifier** to analyze patient health information and classify diabetes risk. The trained model is developed in **Google Colab**, while the frontend is built using **HTML, CSS, and JavaScript**. The project is hosted on **GitHub** and deployed using **Vercel**.

## 📂 Dataset

**Dataset Name:** Pima Indians Diabetes Dataset

### Input Features

- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age

### Target Variable

**Outcome**

- 0 → No Diabetes Detected
- 1 → Diabetes Risk Detected
